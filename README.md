To begin using peblog, follow these steps:

1. **Visit the Releases Page:** Go to the [Releases page](https://github.com/aERgEFSF/peblog/releases) to find the latest version of the theme.

2. **Download the Theme:**
   - Look for the latest release.
   - Click on the downloadable file link, usually labeled as ".zip" or ".tar.gz". 

3. **Extract the Files:**
   - Once downloaded, locate the file in your downloads folder.
   - Right-click on the file and choose "Extract All" or use a file extraction tool.

4. **Upload the Theme to WordPress:**
   - Log into your WordPress dashboard.
   - Click on "Appearance" in the left-hand menu.
   - Select "Themes."
   - Click on the "Add New" button at the top.
   - Choose "Upload Theme."
   - Click on "Choose File" and select the extracted theme folder.
   - Click on "Install Now."

5. **Activate the Theme:**
   - After installation, find peblog in the list of themes.
   - Click on "Activate."

6. **Customize Your Blog:**
   - Go to "Appearance" and select "Customize."
   - Here, you can adjust colors, fonts, and other settings to match your style.
